INSTALL
1. Go to Admin all and to add module definition. 
2. Point to install.xml install file
3. Add the module to a page
4. Edit module settings: See below
5. Use it! ;o) 
Note: The module is automatically installed when you install Rainbow.
The install procedure is only required if you deleted the module in Admin all

Module settings
---------------
Documentpath - the subfolder beneath the 'Site Path' for saving the uploaded files
Show Image column? - Mark this if you like see Image Icons
Show Title column? - Mark this if the title column should be displyed
Show Owner column? - Mark this if the title owner should be displyed
Show Area column?  - Mark this if the title area should be displyed
Show Last Updated column? - Mark this if the Last Updated column should be displyed
